SNOT-22 Decision Tree Estimator

An interactive web app to estimate SNOT-22 total scores using a decision tree model. Designed to reduce survey burden, the tool asks only the most informative questions while maintaining accurate predictions. Users can optionally provide demographics, answer questions via sliders, and view their predicted score along with a table of responses.

The app includes a full SNOT-22 decision tree graphic and runs entirely in the browser.

The decision tree is the Depth-12 CART described in the poster.

Developed by: Katrina Bootes
Model generated with: R (rpart)

Note: For research and demonstration purposes only; not a medical diagnostic tool.
